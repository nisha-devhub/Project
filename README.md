# Failure Reasons for UPI Payment

## Project Overview

This project analyzes the common reasons for UPI payment failures using data visualization. The objective of the project is to identify different factors that cause UPI transactions to fail and present them through an interactive dashboard.

## Tools Used

* Microsoft Power BI
* Data Visualization
* Data Analysis

## Description

UPI (Unified Payments Interface) is widely used for digital payments. However, sometimes transactions fail due to various reasons such as network issues, bank server problems, insufficient balance, or technical errors.

In this project, a dataset related to UPI transactions is analyzed using Power BI. The dashboard helps visualize and understand the main causes of payment failures and the frequency of these issues.

## Dashboard Features

* Visualization of different UPI payment failure reasons
* Charts showing frequency of transaction failures
* Insights into the most common causes of failed payments
* Interactive dashboard for better understanding of the data

## Purpose of the Project

The main purpose of this project is to analyze UPI transaction failures and provide insights that can help improve digital payment systems in the future.
